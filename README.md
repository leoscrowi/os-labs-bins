# `troubleshooting`
Для начала выполнения задания 7.3 (с сигналами), вам потребуется решить проблему с файловыми операциями (см. 7.2)

После решения проблемы с пункта 7.2 программа будет работать 20 секунд, чтобы вы успели отправить нужный сигнал (до решения проблемы с файловыми операциями программа выполняется сразу)
